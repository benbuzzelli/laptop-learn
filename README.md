# Dino Desk

A dinosaur-themed laptop skills training game for 2-4 year olds. Built with React, TypeScript, and HTML5 Canvas.

## Games

- **Egg Hunt** - Tap eggs to hatch baby dinosaurs
- **Dino Path** - Follow the dots to guide a dinosaur home
- **Spell Dino** - Drag letters to spell dinosaur names
- **Volcano Escape** - Help a T-Rex escape the volcano
- **Dino Match** - Flip cards to find matching dinosaur pairs
- **Jungle Explorer** - Find dinosaurs hiding in the jungle

## Features

- Dino Museum collection page to track discovered dinosaurs
- Sticker rewards for milestones
- Easy mode toggle for younger players
- Parent dashboard with play time stats and data reset
- All rendering done on HTML5 Canvas with custom pixel-art sprites

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or later)
- npm (included with Node.js)

### Install and Run

```bash
cd laptop-learn-app
npm install
npm run dev
```

Then open the URL shown in the terminal (usually http://localhost:5173).

### Build for Production

```bash
cd laptop-learn-app
npm run build
npm run preview
```

## Tech Stack

- React 19
