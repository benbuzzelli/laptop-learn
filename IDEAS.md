# Dino Desk – Ideas

Running list of ideas for future development.

## Three-Tier Difficulty System

Replace the current easy mode toggle with **Easy / Medium / Hard** across all games. The parent dashboard should show **recommended ages** for each difficulty tier.

### Per-Game Difficulty Ideas

**Spell Dino**
- Easy: click-letters mode (current easy).
- Medium: type letters on the keyboard (current normal).
- Hard: falling-dinosaur typing game — dinos fall from the top with their names above them and must be typed before they hit the ground.

**Volcano Escape**
- Easy: all-direction hints + guided path (current easy).
- Medium: valid-direction hints (current normal).
- Hard: no hints, no trail, tighter mazes.

**Dino Dungeon** (kept separate from Volcano Escape)
- Already supports Easy mode (slower/shorter-sighted guards). Add Hard mode with faster guards and wider vision.

**Egg Hunt**
- Easy: big hit areas, hints (current easy).
- Medium: standard (current normal).
- Hard: eggs drift/roll around the scene and must be chased down.

**Dino Path**
- Easy: guide line + "Go here!" label (current easy).
- Medium: standard dotted path (current normal).
- Hard: branching paths where only one route reaches home.

**Jungle Explorer**
- No changes planned for now.

**Dino Match**
- Easy: fewer cards (current easy).
- Medium: standard grid (current normal).
- Hard: cards auto-hide after a short preview — you have to remember without seeing them flipped.

## Open Questions
